
# QUICKY 🛒

Quicky is an Android application built as a **college project** to demonstrate e‑commerce style functionality with modern Android development practices.  
It allows users to browse products, add them to cart, and manage purchases with a clean UI.

---

## 📌 Overview
- A simple and intuitive shopping app.
- Built with **Android Studio + Kotlin**.
- Implements **MVVM architecture** for clean separation of logic and UI.
- Includes cart management, product listing, and login flow.

---

## ✨ Current Features
- **Login & Authentication** (basic login screen).
- **Product Listing** with images and details.
- **Cart Management** (add/remove items, view cart).
- **ViewModel Integration** for reactive UI updates.
- **Custom Adapters** for product and cart lists.
- **Material Design UI** with themes and colors.

---

## 🛠️ Technologies Used
- **Kotlin** for Android development.
- **Android Jetpack Components** (ViewModel, LiveData).
- **RecyclerView & Adapters** for product/cart lists.
- **Gradle** for build automation.
- **GitHub** for version control and collaboration.

---

## 🚀 Future Enhancements
- **Firebase Authentication** for secure login/signup.
- **Realtime Database / Firestore** for product storage.
